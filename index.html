ABOUT

What is Scheduling?

Scheduling is an interactive visualizer that demonstrates how CPU scheduling algorithms work — including First Come First Serve (FCFS), Shortest Job First (SJF), and Priority Scheduling. It aims to help students and developers easily understand how processes are executed, waiting times are computed, and how different algorithms affect overall CPU performance.

Features

  • Real-time visualization of CPU scheduling.
  • Animated Gantt Chart for process flow.
  • Step-by-step computation of Average Waiting Time (AWT).
  • Modern light/dark mode for better viewing.
  • Simple and intuitive UI for learning and exploration.

Built With

  • HTML, CSS, JavaScript
  • GSAP Animations for smooth transitions
  • Responsive UI Design for all screens

Educational Purpose

This website was developed as a learning tool for BS Computer Engimeering students to visualize CPU scheduling behavior in real time, making it easier to grasp core Operating System concepts.
